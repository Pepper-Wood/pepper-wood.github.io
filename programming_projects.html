Adoptapedia (Python and PHP)

Adoptapedia - Adoptable Encyclopedia - is an online database to help artists who create and sell character designs called "adoptables". I started this project in 2012 beginning with an excel spreadsheet by copying and pasting group names. Once I began learning to code, I wrote the application first with Java and have since rewritten in Python. The project is currently undergoing development into a full-fledged standalone website found at adoptapedia.com with its respective code repository found here. The current deployed version of this project can be found at adoptapedia.deviantart.com with its respective code repository found here.

Conway's Game of Life (JavaScript, HTML, and CSS)

Fall 2015, this was a group computer graphics project focussed on learning to use various javascript graphics engines. Once initializing the Grid Size and setting cube states to Random, a rotating grid of cubes of random colors spins on the lower part of the screen. These cube states then activate and deactivate following the rules in John Conway's game of life. The button at the top of the screen adds a fun dimension by switching the random colors of the cubes to images of Professional Wrestler John Cena as his theme song plays over. The project can be viewed at this link, with its respective code repository found here. Note: this simulation will only work in Firefox!

FreezeAlert (Python)

Spring 2015, this was a group engineering project to create a solution to pipes freezing and bursting in houses especially during the winter. Our physical units consisted of two raspberry pis - one to act as the central control and the other to be mounted on a pipe in a household with a temperature censor. My contributions was the entirety of the code for the project, which focussed on: enabling communication between these two units, create a tracking graph of the temperature of the pipe over time, and turn on an off a motor to help run water through the pipes and prevent it from being standstill. The simple website I made for this project is still live and can be found here, with its respective code repository found here.
